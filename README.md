## 介绍
- 前后端分离，此项目仅为前端
- 主要是在React中使用TypeScript，使用脚手架搭建react环境
- 模块化，分成多个组件，利用Redux进行通信
- 图表使用svg

## 功能
#### 已完成
- 注册登录登出
- 番茄钟25min
- Todolist 待办任务 增删改功能
- 数据统计
